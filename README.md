# Shoolini

A small React + Vite landing page for an educational or placement-focused service built with Tailwind CSS.

## Features

- Responsive UI with modern components
- Course cards, FAQ section, placement-focused content
- Contact and consultation form
- Simple landing-page structure for a school or training program

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Hook Form
- Lucide Icons
- React Icons

## Scripts

- `npm install` - install dependencies
- `npm run dev` - start development server
- `npm run build` - build production files
- `npm run preview` - preview production build
- `npm run lint` - run ESLint

## Project Structure

- `src/main.jsx` - application entry
- `src/App.jsx` - root component
- `src/Components/` - UI components and page sections
- `src/Pages/Home.jsx` - home page layout

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.

## Notes

This repository is a lightweight front-end project for a small educational landing page.
